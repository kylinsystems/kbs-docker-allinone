2017-11-18 更新无数词条，v5.1版本已测试 

【好消息】 这个语言包已经近乎可以接受了。

【说明】 ****请在测试服务器安装本语言包****
1. 新安装，不用考虑词条冲突，直接使用本语言包
2. 已有自定义语言包，先讨论集中维护词条，比如“有效”，自定义为”是否有效“
  2.1 如果客制词条比较少，并可根据时间区分，可导出客制词条，安装后再客制。
  2.2 如果客制词条比较多，又希望导入新语言包更新，可分表进行name比较，然后根据结果合并
  2.3 建议使用临时表或者视图来对比已有自定义词条
3. 本语言包未修改系统默认的非集中维护选项，故不会影响客制化非集中词条。
   虽然系统需要进行很多非集中设置，比如”支付“改为”付款“或者”收款“，但本语言包不含sql，无
   法修改ad字典的配置（下个版本完善）。
4. 为什么每次都和自用语言包的冲突那么大，怎么解决？
  4.1 免费语言包发布太慢了，只能自己更新，但未回馈贡献到免费语言包，反馈就不会再有冲突
  4.2 语言包能不能及时更新？其实你也可以更新。
  4.3 我可以帮你做。
5. 下次什么时候更新语言包？
  5.1 你发布一篇idempiere有关的博客，联系我，我就发送一个最新的语言包给你。
